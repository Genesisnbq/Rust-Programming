# Power of vscode

> 一些个人比较常用的vscode快捷操作

1. `command + F12` 以预览方式在当前页面显示定义
2. `Shift + F12` 以预览方式在当前文件打开引用的文件列表
3. `command + p 输入 @` 显示大纲（符号）
4. `command + shift + p` 输入 `reload window` 重载vscode
